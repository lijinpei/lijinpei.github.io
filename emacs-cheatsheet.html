C-f C-b C-n C-p C-a 行首 C-e 行尾

M-f/C-RIGHT/M-RIGHT M-b/C-LEFT/M-LEFT 以单词为单位移动 M-r/move-to-window-line-top-bottom 无参数时，连续M-r，不移动text，在中间最左，顶部最左，底部最左之间切换。有参数时，移动到相应的screen line最左，非负数从屏幕头部数，负数从底部数。 M-< 移动到buffer的开头。数字参数n表示n/10。 M-> 移动到buffer的末尾。

C-v/PageDown/next 向下翻页 M-v/PageUp/prior 向上翻页 M-g c 读取一个数字参数n，移动到第n个字符。 M-g M-g/M-g g 移动到第n行。n可以读取数字参数;若point在一个数字上后数字之后，RET使用这个数字;或者通过前缀参数。第一行编号是1。 M-g tag 移动到第n列。n可以是读取数字，或者数字前缀参数。 C-u C-x C-n Cancel the goal column. Henceforth, C-n and C-p try to preserve the horizontal position, as usual.

DEL是BACKSPACE。 Delete是键盘上写着Del的键。

DEL/BACKSPACE/delete-backward-char 向后删除(字符或region)。 Delete/delete-forward-char 向前删除(字符或region)。 C-d/delete-char向后删除（仅字符）。 C-k/kill-line 删除到行末。 M-d/kill-word 删除到下个单词的末尾。 M-Del/M-BACKSPACE/backward-kill-word

c-//c-x u/c-_/undo C-x C-f 打开文件。 C-x C-s 保存文件。 C-h k + 按键 describe-key 给出关于按键的帮助。 C-h/F1 帮助命令。

C-o/open-line 光标后插入空行。 C-x C-o/delete-blank-lines 删除连续空行，只保留一个。 若在连续的空行上，则只保留一个空行。 若在单个空行上，则删除相应空行。 若在非空行上，删除下面的所有空行(只包括紧跟着的空行)。

visual line mode 在word boundary wrap line，C-a, C-n, C-k以visual line为单位。

M-x what-line 显示point的行号。 M-x line-number-mode 在mode-line中显示行号。 M-X column-number-mode 在mode-line中显示列号。 M-=/count-word-region 显示region中的行数，单词数，字符数。 C-u M-= 显示整个buffer的行数，单词数，字符数。 M-x count-words 显示行数，单词数，字符数。如果有region active，则显示region的;否则显示buffer的。 C-x =/what-current-position 显示关于当前point位置，以及point后字符的信息。 选项what-cursor-show-name 显示字符的名字。 M-x hl-line-mode 启用高亮当前行模式。 M-x size-indication-mode 启用自动显示当前buffer的大小模式。 display-line-numbers-mode

numeric-argument/prefix-argument

方法一: M-数字1 数字2 数字1可以是负号 C-u可用来显式结束序列 方法二: C-u/universal-argument 单独C-u表示4个，连续C-u表示4的幂。 C-u + 数字也可以用C-u显式结束序列。

还有一种参数叫minibuffer argument

On graphical displays, C-0, C-1, etc. act the same as M-0, M-1, etc.

C-x z/repeat 重复命令，不重新读取参数。z的个数表示重复次数。
